ItemsToMap
ItemsToMap(inputProp, keyProp: string, valueProp?)

Permite generar un mapa de datos a partir del arreglo de elementos de la propiedad inputProp

Parameters :
Name Type Optional Description
inputProp No

Propiedad de tipo array sobre la qeu se debe generar el map,
keyProp string No

Propiedad dentro de cada elemento del array que se utilizara como llame del map, por defecto es 'id'
valueProp Yes

Propiedad dentro de cada elemento qeu se desea guardar en el mapa, es opcional, si no se especifica el mapa se guardara con referencia a los objetos completos
Example : 
propertyName = &#39;itemsMap&#39; , toma valor &#39;items&#39; para inputProp
propertyName = &#39;itemsmap&#39; , toma valor &#39;items&#39; para inputProp
